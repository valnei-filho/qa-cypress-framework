# QA Cypress Framework

Automation testing framework developed with **Cypress**, following industry best practices such as **Page Object Model (POM)**, reusable methods and organized project structure.

---

## 🚀 Technologies

- Cypress
- JavaScript (ES6+)
- Node.js
- Git
- GitHub

---

## 📁 Project Structure

```
cypress
│
├── e2e
│   └── login.cy.js
│
├── fixtures
│   └── data
│
├── pages
│   └── LoginPage.js
│
└── support
    ├── commands.js
    └── e2e.js

cypress.config.js
package.json
```

---

## ✅ Implemented Features

- Page Object Model (POM)
- Organized selectors
- Successful login test
- Invalid login test
- Reusable methods
- Clean project structure

---

## ▶️ Running the project

Clone the repository

```bash
git clone https://github.com/valnei-filho/qa-cypress-framework.git
```

Install dependencies

```bash
npm install
```

Run Cypress

```bash
npx cypress open
```
